# SNSテンプレ

ストリーマーマガジンのX（Twitter）投稿用インフォグラフィック画像のテンプレートです。

## 使い方
1. Claudeに記事のURLを渡す
2. このHTMLのデザインに合わせてインフォグラフィック画像を作成してもらう
3. 「LGTM」と伝えるとXの投稿案を3つ出してもらえる
4. HTMLをスクショして投稿画像として使用する

## デザイン仕様
- サイズ：1200×900px（横長3:4）
- 背景：方眼ノート風
- カラー：ピンク・ブルー・グリーン

## プロンプト例

### プロジェクト指示文
投稿用インフォグラフィック画像とXの投稿文を作るプロジェクトです。

添付のHTMLファイルのデザイン・レイアウトを必ず参考にしてください。
記事のURLや内容を渡したら、そのデザインに合わせてHTMLで画像を作成してください。

【進め方】
1. まずHTMLの画像を作成する
2. 私が「LGTM」と言ったら、Xの投稿案を3つ出す
3. それ以外のタイミングでは投稿案を出さない

【Xの投稿文について】
- フレンドリー・カジュアルなトーン
- すでに配信している人向け
- 記事の内容に軽く触れながらまとめる
- ハッシュタグあり
